## ❓ Travelog 소개
### 💡 기획 의도
- 지도를 보면서 여행 계획을 세워보자!
- 어디론가 떠나고 싶을 때... 추천받고 싶어
- 나만의 여행 추억을 기록으로 남겨보자
- 나만 볼 수 없지! 기록을 공유하고 싶어!

### 📸 주요 기능
- 🔎 여행 계획
    - 여행지와 일정을 선택
    - 지도를 보며 가고 싶은 장소 결정
    - 순서도 마음대로 바꿀 수 있도록
- 🍔🍖 목적지 추천
    - 관심가는 여행지를 선택
    - 여행지의 볼거리, 먹거리, 축제 등을 확인
- ✏ 여행 기록
    - 여행 후 사진과 함께 기록 남기기
    - 지도와 함께 여행 되돌아보기
- 🤝 기록 공유
    - 여행 기록을 다른 사람과 공유
